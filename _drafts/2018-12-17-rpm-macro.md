# rpm宏

> 由于最近在打包一个rpm的时候发现这个包过于庞大，编译时间非常长，因此想要添加-j参数提升编译速度。

## 查看rpm宏

​	查看rpm宏的命令非常简单：

``` shell
$ rpm --eval %{_sysconfdir}
/etc
```

​	这是查看某个特定的宏；接下来看所有的宏：

``` shell
$ cat /usr/lib/rpm/macros
```

​	这个文件中就包含了所有的rpm宏啦。

